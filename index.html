News

All original features done!

All SBot Original Java bot features have been ported, with the exception of Repeat and Metro Times. We hope to bring Metro Times soon, along with some non-original LEAGUE Bot features (the original SBot was built in association with the LEAGUE of Amazing Programmers and therefore has some other, non-original features).

Commands

!beep Get the bot to beep!
!analogy [measurement] Get a useless analogy for the given measurement.
!api [url] GET any RESTful JSON endpoint.
!dynamiclink [link] Shorten the given link.
!comadd [item] Store an item in the community database.
!comget [key] Get an item in the community database.
!blogger [id] Get the latest post from any Blogger blog, by ID.
!github [user] Get all a user/org's repos.
!story Toggle long story mode.
!name Generate a random name.
!ctatt [mapid] Get the next train time. Buses coming soon.
